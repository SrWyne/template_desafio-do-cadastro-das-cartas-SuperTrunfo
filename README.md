===============================
JOGO SUPER TRUNFO DE CIDADES
===============================

DESCRIÇÃO:
Este programa simula um jogo de Super Trunfo utilizando cidades.
O usuário cadastra duas cartas contendo informações da cidade e pode comparar atributos.

--------------------------------
ATRIBUTOS DAS CARTAS:
--------------------------------
- Estado
- Código
- Nome da Cidade
- População
- Área (km²)
- PIB
- Pontos Turísticos
- Densidade Populacional
- PIB per Capita

--------------------------------
FUNCIONAMENTO:
--------------------------------
1 - O usuário escolhe iniciar o jogo
2 - Preenche os dados da Carta 1
3 - Preenche os dados da Carta 2
4 - Pode escolher:
    - Mostrar todos os atributos
    - Comparar um atributo específico
5 - O sistema informa a carta vencedora

--------------------------------
REGRAS:
--------------------------------
- Todos os campos devem ser preenchidos
- Valores numéricos não podem conter letras
- Em caso de empate, o sistema informa

--------------------------------
COMO COMPILAR:
--------------------------------
gcc nome_do_arquivo.c -o jogo

--------------------------------
COMO EXECUTAR:
--------------------------------
Windows:
jogo.exe

Linux/Mac:
./jogo

--------------------------------
OBSERVAÇÕES:
--------------------------------
- O programa utiliza estruturas condicionais (if/else)
- Utiliza switch case para controle de menu
- Utiliza fgets para leitura de strings com espaço
- Utiliza cálculos matemáticos para comparação
